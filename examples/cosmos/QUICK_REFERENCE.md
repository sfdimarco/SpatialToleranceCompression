## ⚔️ Combat Basics

1. **Select** your system (blue glow + orbit rings)
2. **Click** enemy system to attack
3. **Fleet travels** along hyperlanes (white lines)
4. **Battle resolves** on arrival:
   - If defender HP ≤ 0: **System captured!**
   - If attacker fleet destroyed: **Attack failed**
   - Both weakened: **Stalemate**

**Pro Tip**: Send fleets when you have 20+ ships for reliable captures.

## 💎 Commander Abilities

### [1] Orbital Strike 🌠
- **Effect**: Deal 50 damage to selected enemy system
- **Cooldown**: 60 seconds
- **Best Used**: Finishing low-HP targets, weakening fortifications

### [2] Reinforce Fleet 🚀
- **Effect**: Add 20 ships to selected system
- **Cooldown**: 45 seconds
- **Best Used**: Reinforcing border systems, preparing big attacks

### [3] Deep Scan 📡
- **Effect**: Reveal all systems and their stats
- **Cooldown**: 30 seconds
- **Best Used**: Early game scouting, finding weak points

### [4] Emergency Shield 🛡️
- **Effect**: Shield all player systems (100 HP)
- **Cooldown**: 90 seconds
- **Best Used**: Defending against multiple attacks

## 🔬 Technology Tree

Press **T** to view tech tree.

### Tier 1 (50 points each)
- **Advanced Hulls**: +25% System HP
- **Fleet Logistics**: +50% Fleet Speed
- **Resource Scanners**: +30% Resource Gain

### Tier 2 (100 points each)
- **Plasma Weapons**: +20% Battle Damage
- **Shield Generators**: Regenerate 5 HP/tick
- **Warp Drives**: +40% Fleet Range

### Tier 3 (200 points each)
- **Dreadnought Class**: Spawn powerful dreadnoughts
- **Orbital Platforms**: +50% Defense
- **AI Warfare**: Auto-battle advantage

**Earn Tech**: Capture research stations (purple systems)

## 🎯 Strategy Tips

### Early Game (Tick 0-200)
✅ Capture nearby neutral systems first  
✅ Grab resource systems (green)  
✅ Use abilities frequently (they'll recharge)  
✅ Get "First Blood" bonus (<200 ticks)  

### Mid Game (Tick 200-1000)
✅ Fortify border systems with enemy neighbors  
✅ Build up fleet before attacking  
✅ Research Tier 1 tech for bonuses  
✅ Watch for pirate raids (tick 100, 300, 500...)  

### Late Game (Tick 1000+)
✅ Coordinate multi-front attacks  
✅ Use combo system for max points  
✅ Save abilities for crucial moments  
✅ Prepare for AI "last stand" bonuses  

### AI Personalities
- **Red (Aggro)**: Constant attacks, predictable. Counter-attack when weak.
- **Yellow (Econ)**: Rich but militarily weak. Pressure early.
- **Green (Defense)**: Fortified, hard to dislodge. Starve them instead.

## 🎪 Combo System

Build combos by capturing systems quickly:

| Combo Level | Bonus |
|-------------|-------|
| x1.0 | Base |
| x1.5 | 5+ captures |
| x2.0 | 10+ captures |
| x2.5 | 15+ captures |
| x3.0 | 20+ captures (max) |

**Combo decays** over time—keep capturing to maintain!

## ⚠️ Common Mistakes

❌ **Overextending**: Capturing too many distant systems  
→ Leaves you vulnerable to counter-attacks  

❌ **Emptying Systems**: Sending all ships on one attack  
→ Leaves home defenseless  

❌ **Ignoring Tech**: Not researching upgrades  
→ Fall behind in combat effectiveness  

❌ **Wasting Abilities**: Using abilities without purpose  
→ They'll be on CD when you really need them  

❌ **Turtling**: Staying in starting system  
→ You'll be outpaced economically

## 💡 Pro Tips

1. **Expand in pairs**: Capture two systems at once for mutual defense
2. **Use hyperlanes**: Fleets move faster along connections
3. **Watch the minimap**: See enemy fleet movements
4. **Save one ability**: Always keep an ability available for emergencies
5. **Research economy first**: Resource bonus pays off long-term
6. **Attack when AI fights**: Strike while enemy is weakened
7. **Combo before big push**: Build combo, then capture multiple systems

## 🐛 Troubleshooting

**Game too hard?**
→ Use abilities more! They're powerful.

**Game too easy?**
→ Try capturing capitals last, build score first.

**Performance issues?**
→ Press M/E/H to disable panels, reduce speed with -.

**Stuck?**
→ Press R to restart with new galaxy.
